## 💡 学習のコツ

### 1. 実際に手を動かす

```bash
# ❌ 読むだけ
# ✅ 実際にコマンドを実行する

# 例：最初のコマンド
aws sts get-caller-identity
```

---

### 2. エラーを恐れない

```bash
# エラーメッセージは親切
$ aws ec2 describe-instances --region invalid-region

# エラー内容をよく読む
# → リージョン名が間違っていることがわかる
```

---

### 3. helpを活用

```bash
# サービスのヘルプ
aws ec2 help

# コマンドのヘルプ
aws ec2 describe-instances help

# 必ず --help または help をつける習慣を
```

---
